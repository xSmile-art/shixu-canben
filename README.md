# Webnovel Workshop

网络小说创作工作区。当前在写长篇都市异能《时序残本》（悬疑解密·智斗流，男频，300万字+规划）。

## 目录结构

```
webnovel/
├── novels/
│   └── 时序残本/
│       ├── 00-记忆台账.md          # 全书唯一权威记忆源（剧情/人物/伏笔状态）
│       ├── 01-立项卡.md            # 立项：一句话pitch+核心钩子+差异化亮点
│       ├── 02-世界观.md            # 七层架构世界观设定
│       ├── 03-人物卡.md            # 主角+配角+反派卡
│       ├── 04-大纲.md             # 三级大纲（全书主线+卷一+逐章）
│       ├── 08-人物关系台账.md      # 人物关系总表+冲突绑定+动态演进
│       ├── 09-伏笔台账.md          # 三级伏笔登记与回收追踪
│       └── 正文/                  # 逐章正文
│           ├── 第001章-草稿层.md
│           ├── ...
└── .gitignore
```

## 创作规范

遵循 `webnovel` skill 的模块化规则：立项卡 / 世界观 / 人物卡 / 大纲 / 正文 / 关系台账 / 伏笔台账 / 记忆台账 八件套，每次创作后同步台账。

## 在线阅读

部署在 GitHub Pages：https://xsmile-art.github.io/shixu-canben/

### 加新章节

1. 写 `novels/时序残本/正文/第NNN章-标题.md`
2. 在 `chapters-index.json` 追加 `{"num":"NNN","title":"标题"}`
3. `git push origin main`

网页在 raw CDN 刷新后（最多 5 分钟）即可读到新章，无需重新构建站点。React 阅读器代码在 `reader/`，改代码才会触发部署。
