# Creative Memory Module

## Core Purpose

This file is the single authoritative memory ledger for the entire book. All module creation must prioritize loading this file. All setting, plot, and character changes must be synced to this file. The latest version of this file is always the source of truth.

## Execution & Sync Rules

1. Before starting any creative task, you must load this ledger first and verify all core information
2. After completing any creative work, you must sync the corresponding sections to keep the ledger current
3. Core settings (worldview foundational rules, protagonist core profile, main plot) must not be arbitrarily modified — if modification is necessary, annotate the reason and version number
4. All new settings must be entered into this ledger before being used in chapter writing — no ad-hoc settings
5. Conduct a ledger review at the end of each volume — clean up stale information and fill in gaps
