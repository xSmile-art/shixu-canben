# Review & Analytics Module

## Execution Flow

1. Collect core data for the target chapter range
2. Analyze data across multiple dimensions
3. Identify problematic chapters and root causes
4. Output optimization plan and direction for adjustments
5. Sync planning updates to the creative memory ledger

## Core Data Dimensions

- **Retention Rate (追读率 / zhuīdúlǜ)**: The core metric — reflects a chapter's ability to retain readers
- **Completion Rate (读完率 / dúwánlǜ)**: Reflects chapter pacing and engagement
- **Comment Count (评论数 / pínglùnshù)**: Reflects plot's ability to provoke reader emotion
- **Demand-for-More Count (催更数 / cuīgèngshù)**: Reflects hook strength and plot appeal
