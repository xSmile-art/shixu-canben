# Character Design Module

## Execution Flow

1. Design the protagonist's core profile based on the premise and worldbuilding
2. Design key supporting characters and antagonists
3. Map out character growth arcs
4. Output character cards
5. Validate against the checklist
6. Sync to creative memory ledger and character relationship ledger

## Character Validation Checklist

- [ ] Protagonist profile is distinctive, memorable, not flat
- [ ] Protagonist has clear flaws and limitations; not a perfect saint
- [ ] Every supporting character has a clear narrative function; no pure tools
- [ ] Antagonist motivation is reasonable and behavior is logically consistent
- [ ] Character growth arcs are clear and tied to the main plot
- [ ] Synced to creative memory ledger and character relationship ledger
