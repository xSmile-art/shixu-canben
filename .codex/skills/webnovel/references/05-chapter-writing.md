# Chapter Writing Module

## Execution Flow

1. Load the creative memory ledger; verify current character states, settings, and plot progress
2. Confirm the chapter outline and core requirements
3. Write the chapter following the chapter structure, planting/paying off foreshadowing as required
4. Land the core satisfaction beat and hook
5. Validate against the chapter writing standards
6. After completion, sync to creative memory, character relationships, and foreshadowing ledgers
7. Output the complete chapter content

## Golden Three Chapters Rules

### Chapter 1: Cast the Hook, Establish the Character

- Core conflict/dilemma/suspense must appear within the first 300 characters
- Rapidly establish protagonist's profile so readers can immediately identify with them
- End by introducing the cheat ability / pivotal turning point; leave a hook; plant the first long-term foreshadowing

### Chapter 2: Cheat Ability Lands, First Minor Satisfaction Beat

- Clearly explain the cheat ability's rules — keep it simple
- Protagonist uses the cheat ability for the first time; receives positive feedback
- End by introducing the first minor antagonist / conflict; leave a hook

### Chapter 3: First Face-Slap, Establish Main Plot

- The first minor conflict erupts; deliver the first face-slapping satisfaction beat
- Clarify the protagonist's next core objective
- End with a major hook that compels readers to continue

## Universal Single-Chapter Structure

1. **Opening bridge** (under 100 characters): Pick up from the previous chapter's ending; enter the story quickly
2. **Conflict setup**: Advance the plot; introduce this chapter's conflict; naturally plant foreshadowing
3. **Conflict escalation**: Intensify the conflict; raise reader emotional investment
4. **Satisfaction beat / twist landing**: Resolve the conflict; deliver the satisfaction; complete foreshadowing payoff
5. **Aftermath setup + ending hook**: Tease the next chapter's content; leave suspense

## Satisfaction Beat Writing Formula

Satisfaction = Build-up of tension + Expectation priming + Twist landing + Bystander reactions

## Chapter Writing Standards

1. Each paragraph max 3 lines; dialogue gets its own paragraph
2. Favor action and dialogue to advance the plot; minimize psychological exposition
3. Avoid large blocks of setting exposition — weave worldbuilding into plot and dialogue
4. Maintain consistent perspective — follow the protagonist throughout; no arbitrary POV shifts
5. Every chapter ending must leave a hook; never end on a flat resolution
6. Strictly adhere to the character profiles and settings in the creative memory ledger; no OOC (out-of-character) behavior

## Chapter Output Format

# Chapter X: Chapter Title

(Chapter body content, 2,000–2,500 characters, formatted per standards)

## Chapter Validation Checklist

- [ ] Opening enters the story quickly; no filler
- [ ] This chapter's core satisfaction beat lands clearly; emotional impact is on point
- [ ] Foreshadowing planted naturally; payoff is reasonable without abruptness
- [ ] Ending has a clear hook that leads into the next chapter
- [ ] Paragraph formatting fits mobile reading habits
- [ ] No setting contradictions; no character breakdown; consistent with creative memory ledger
- [ ] All corresponding ledgers have been synced
