# 世界观设定 · 时序残本

> 本文件为《时序残本》世界观档案，遵循七层架构。核心设定以记忆台账（00-记忆台账.md）为准，二者冲突时以台账为准并记录版本。

## Layer 1：基本规则

- **核心世界前提**：异能者早已存在并被官方系统性隐藏。社会表层运转如常，地下潜伏着异能黑市、犯罪组织与官方特管机构。这是一座"看起来正常、实则暗流分层"的现代都市群。
- **规则起源**：异能源于"现实稳定性"的局部松动——某类人能感知并干预现实的"草稿层"，把可能性凝固成既定事实。时间系异能是其中最稀有、最危险的一支，触及因果本质。
- **对普通人的影响**：绝大多数人终其一生不会察觉异能存在；偶有目击会被官方"秩序修复"（记忆淡化 + 现场重建为意外）。普通人活在"被校准过的现实"里而不自知。

## Layer 2：地理与势力

- **地理布局**：故事以虚构沿海都市「临港市」为主舞台（金融繁华、跨海大桥、旧工业区、地下管网密布），后续扩展至区域城市群与境外暗点。
- **核心势力划分**：
  - **官方 · 锚管局**（秩序锚管理局）：负责异能者登记、维稳、灭口善后；维护"现实秩序"，对外不存在。
  - **地下 · 蚀刻会**：追求"时间本质"的神秘组织，收割时间系觉醒者，意图改写历史稿本；本作长线反派核心。
  - **灰色 · 游荡者**：不站队的散修式异能者，黑市情报/走私的中转节点。
  - **民间黑市**：异能物品、伪异能药剂、情报交易的灰色市场。
- **势力等级基准**：锚管局与蚀刻会均以"现实扭曲量级"评估威胁，等级越高追剿/收割优先级越高。

## Layer 3：力量 / 异能体系

异能者统称「特异者」。分级按"对现实稳定性的扭曲量级"划定，与官方/地下社会地位挂钩。

### 核心等级结构（由低到高）

| 等级 | 代号 | 核心能力 | 对应社会地位 |
| ---- | ---- | -------- | ------------ |
| 1 | 潜流 | 单一低危异能，常规武器可制服 | 黑市底层 / 混迹平民 |
| 2 | 裂隙 | 异能稳定，小范围扭曲现实 | 灰色职业 / 黑市骨干 |
| 3 | 断层 | 可影响数十人 / 小区域 | 官方特管员 / 组织中层 |
| 4 | 汐位 | 可影响大区域 / 城市级 | 官方高层 / 组织首脑 |
| 5 | 锚点 | 概念级，锚定某条现实规则 | 极少数，传说级 |
| 6 | 织时 | 涉及时间 / 因果本质 | 禁忌，被视为神话 |

### 异能获取与晋升

- **觉醒方式**：多为"濒死/极端冲击"诱发觉醒；少数为血统/污染源诱发。时间系觉醒概率极低且高危险。
- **晋升条件**：加深对"现实草稿层"的干预深度与精度；纯堆次数无效，靠"理解规则"突破。
- **限制与副作用**：
  - 越级干预现实会留下"现实磨损"，被官方监测网标记；
  - 时间系每次回溯/改动叠加"观察者效应"，渗出时间裂缝，可被猎时者追踪；
  - 时间系觉醒者长期回溯会产生"记忆残本污染"——分不清哪条时间线是真经历，长线导致人格磨损（主角核心代价伏笔）。

### 主角能力「残本」成长三阶

- **阶段一 · 倒带（卷一）**：回溯 ≤10 秒，保留记忆；残本仅可"读"不可"用"。
- **阶段二 · 校样（中段）**：可回溯分钟级；调取残本信息做"交叉验证"，对比不同时间线细节差异，还原被篡改/隐藏的真相。
- **阶段三 · 拼贴（后期）**：可在残本间"借力"，把多条时间线的现实片段临时择优叠加到当前，代价极高（现实磨损 + 记忆污染激增）。

## Layer 4：社会与日常

- **普通人日常**：照常通勤、消费、刷手机；"意外事故"新闻里藏着被修复的异能事件。
- **特异者生存规则**：不暴露、不越级、不碰时间系红线；登记在册者受锚管局监控，未登记者随时可能被"修复"。
- **相关职业**：异能物品走私贩、情报掮客、地下"清道夫"（处理异能现场）、官方特管员、秩序修复员、事故重建师（主角职业——天然接触"被修复"过的可疑现场）。

## Layer 5：历史与背景

- **关键世界里程碑**：
  - 数十年前"第一次现实松动"事件，官方与首批觉醒者达成隐匿契约，锚管局前身成立；
  - 其后每隔数年发生一次"局部现实崩塌"被压下，史书里写成天灾/事故。
- **当代背景事件**：近一年临港市"意外事故"异常上升，官方内部怀疑有组织在系统性制造异能事件（暗线，卷一埋）。
- **隐藏世界真相（长线伏笔）**：现实本身像是"被反复改写的稿本"，蚀刻会相信存在某位「执笔人」级存在，而主角的"残本"能力与执笔人之秘存在深层关联。

## Layer 6：剧情相关特殊设定

- **关键秘境/副本**：旧工业区地下"草稿层裂口"（现实稳定性最薄弱处，黑市与蚀刻会交汇点）；跨海大桥"时间锚点"（主角首次觉醒地，后续反复回访）。
- **特殊物品/器物**：
  - 「时序残页」：凝固了某条已逝时间线片段的稀有物，可短暂读取已发生事件的信息残影；
  - 「锚定仪」：官方监测时间裂缝的设备，主角回溯次数越多越容易被它锁定。
- **故事专属规则**：
  - "最小改动原则"——回溯后改动越少，裂缝越小，越难被追踪（主角智斗底层逻辑）；
  - "残本不可逆"——已被改写掉的旧时间线无法当真复活人，只能读取信息（情感与道德张力来源）。

## Layer 7：禁忌与边界

- **世界绝不触碰的规则**：不写真正复活死者（残本只能读信息，不能真把人捞回来）；不写主角靠无限回溯堆次数碾压一切（观察者效应是硬天花板）。
- **上限与天花板**：织时级为神话上限，全书正典角色最高触及锚点级；主角终点为"接近织时的残本执掌者"，不真正成神。
- **禁止能力/行为**：禁止无代价无限回溯；禁止主角用回溯实施违反公序良俗的行为（如反复回溯伤害他人取乐）——此为 OOC 与平台红线双重禁区。
