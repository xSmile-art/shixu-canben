# 大纲系统 · 时序残本

> 本文件为《时序残本》大纲档案（三级大纲）。伏笔节点以伏笔台账（09-伏笔台账.md）为准。本文件与记忆台账（00-记忆台账.md）保持同步。

## 第一级：全书主线大纲

### 核心主线（一句话）

事故重建师沈砚觉醒时间回溯残本，从被官方低估的潜流一步步揭穿潜藏的异能暗世界，最终在与意图改写历史的「蚀刻会·执笔人」的时间博弈中，选择"承担当下"而非"无限改写"。

### 三幕总体结构

#### 第一幕：入局（第 1–2 卷，约 60 万字）

- **核心剧情**：沈砚为恩师之死做事故重建时觉醒残本，半入异能世界；以"编外协查"身份与锚管局苏鸢合作，卷一智斗蚀刻会下层猎时者刻钟。
- **核心爽点**：第一次倒带反杀灭口者（公开式拆穿）、被判定低危潜流实则越级反杀打脸、靠残本交叉验证破解"不可能犯罪"。
- **长线伏笔种植节点**：林叙白真实身份、现实是"被改写稿本"之秘、执笔人、残本污染人格磨损代价、局内鬼。
- **结束节点**：沈砚以编外身份正式半入局，但卷末刻钟案牵出"稿本"线索，直面第一条指向蚀刻会核心的暗线——第一个大危机。

#### 第二幕：成长与扩张（第 3–6 卷，约 180 万字）

- **核心剧情**：残本能力进阶到"校样"与"拼贴"，沈砚在锚管局、黑市、蚀刻会三方间游走扩张影响力，逐层撕开"现实是稿本"的真相，发现"执笔人"的存在。
- **核心爽点**：越级反杀、身份反转（被低估→概念级潜力公开）、连续打脸、用残本做"时间线版本对比"碾压式解谜。
- **长线伏笔回收节点**：林叙白身份揭晓、局内鬼被揪出、刻钟临终线索指向蚀刻会高层。
- **结束节点**：沈砚确认执笔人即终极反派，且自己的残本是其计划关键拼图——全书最大转折。

#### 第三幕：高潮与收束（第 7–10 卷，约 60 万字）

- **核心剧情**：沈砚与执笔人展开跨时间线的终极博弈，解决全部伏笔，作出"承担当下 / 无限改写"的终极取舍。
- **核心爽点**：终局对峙、身份彻底揭穿、回归日常式收束。
- **全部伏笔最终回收节点**：见伏笔台账终局节点。
- **最终结局**：沈砚放弃"无限改写"的诱惑，以"承担当下"破局；现实稿本被定稿，潜藏世界仍隐于都市表层之下——沈砚成为"接近织时"的残本执掌者，回归事故重建师本业，开放式余韵收束。

### 全书爽点里程碑

1. **开篇微高潮（第 3 章）**：第一次倒带反杀灭口者，"对方也在感知时间"钩子砸下。
2. **第一卷大高潮（约第 100 章）**：刻钟被公开式智斗击破，留下指向蚀刻会线索；主角能力被公开判定为"低危潜流"实则越级。
3. **中段大高潮（约第 300 章）**：林叙白真实身份与局内鬼同时揭晓，残本进阶到"拼贴"；世界真相一角撕开。
4. **终局大高潮（结局）**：与执笔人的时间线博弈，"承担当下"一击定稿。

## 第二级：第一卷分卷大纲

- **卷名**：坠桥稿本
- **卷字数**：约 25–30 万字（约 100–120 章）
- **卷核心主题**：从"只信数据的普通人"到"接受异能世界并为之负责"的觉醒与入局。
- **卷核心冲突**：查清林叙白之死真相 vs 蚀刻会灭口者刻钟的三轮智斗。
- **卷起始状态**：沈砚为恩师坠桥意外做事故重建，发现现场物理矛盾。
- **卷结束状态**：刻钟案告破、沈砚以编外协查身份半入局、卷末"稿本"线索浮现。

### 卷内爽点节点（3–5）

1. 第一次倒带反杀灭口者（第 3 章）
2. 残本交叉验证破解"不可能犯罪"打脸警方初步结论（约第 40 章）
3. 被官方判定低危潜流后越级反杀刻钟爪牙（约第 70 章）
4. 卷末公开式智斗擒获刻钟、留下蚀刻会线索（约第 100–110 章）

### 卷内伏笔计划

- **埋**：林叙白真实身份、"稿本"概念、执笔人、局内鬼、残本污染代价、姜十七旧账、跨海桥"时间锚点"
- **收**：林叙白坠桥之谜（揭为灭口非意外）、刻钟案明线（卷内闭环）

### 卷末钩子

刻钟被擒临终/被审讯时吐出一句："你以为这是唯一一版？真正的稿子，在执笔人手里。"——直指第二卷。

## 第三级：首卷逐章大纲（第 1–10 章 · 黄金三章 + 入局推进）

> 第 1–3 章为"黄金三章"，决定追读率，严格遵循：每章前 300 字落钩、每章末留钩、3 章内必落首个微高潮。

### 第 1 章：草稿层

- **本章目标**：建立主角职业与人设；抛出"坠桥现场物理矛盾"钩子。
- **出场人物**：沈砚、（电话中的）林叙白遗属。
- **埋伏笔**：林叙白反复强调的"物理定律不会骗人"；桥面两处不可能共存痕迹。
- **收伏笔**：无（开篇）。
- **剧情流**：
  1. 开篇桥（承接上章钩子，<100 字）：深夜跨海桥，沈砚独自复盘恩师"坠桥意外"，现场数据在掌上终端里对不上。
  2. 冲突发展：两处物理上不可能同时存在的痕迹——护栏擦痕方向与坠落轨迹矛盾，像被"两次写入"过；遗属来电催结案定性为意外。
  3. 爽点/反转落地：沈砚决定独自深夜重返桥面取证——"我重建事故十二年，没还原不出来的案子。"
  4. 结尾钩子：身后有人。黑影一推，他坠下桥。
- **本章核心爽点**：冷峻人设 + 职业自信 + 突发灭口，制造强悬念。
- **本章结尾钩子**：坠落。风。然后——他睁开了眼。
- **埋伏笔**：坠落前最后看见桥面那个"不像意外"的站位。

### 第 2 章：十秒

- **本章目标**：第一次「倒带」觉醒；砸下"对方也在感知时间"核心钩子。
- **出场人物**：沈砚、灭口者（黑影/未露脸）。
- **埋伏笔**：倒带后旧时间线的"残本"第一次留下淡淡残影（主角自己尚不自觉）；桥面站位"预知式"。
- **收伏笔**：坠落死亡（被改写）。
- **剧情流**：
  1. 开篇桥：沈砚从坠落前 10 秒重新"醒来"，站在桥面，灭口者正从背后靠近。
  2. 冲突发展：他本能侧身躲过一推、翻过护栏下滑藏进桥腹钢架；第一次亲历"重来"，以为是濒死幻觉。
  3. 爽点/反转落地：爬回桥面，看见灭口者还站在原位——**像是早知道他会回来，在等他回溯完。**
  4. 结尾钩子：灭口者开口："又倒带了一次？裂缝越来越明显了。"
- **本章核心爽点**：第一次倒带 + 双向时间感知的猫鼠悬念砸场。
- **本章结尾钩子**：反派感知到回溯——"时间不是只有你一个人能动。"

### 第 3 章：残本

- **本章目标**：首卷首个微高潮——靠残本信息反杀灭口者（或逼退）；确立"最小改动"智斗基调。
- **出场人物**：沈砚、灭口者。
- **埋伏笔**：主角意识到"改得越多越被追踪"——观察者效应初现；残本留痕可被读取。
- **收伏笔**：第 1–2 章桥面"不可能痕迹"——残本里显示那是上一版时间线留下的擦痕（现实磨损证据）。
- **剧情流**：
  1. 开篇桥：灭口者逼近，沈砚无路可退。
  2. 冲突发展：沈砚回溯 10 秒、再 10 秒，每次只做"最小改动"——挪一寸站位、留一枚纽扣证据；他注意到旧时间线留下了"残本"残影，可被读取。
  3. 爽点/反转落地：沈砚公开式拆穿灭口者站位——"你站的位置，是只有知道我会回溯的人才会站的位置。"——以"你才是那个感知时间的人"反将一军，迫使对方投鼠忌器退走，留下一句"蚀刻会会找上你的"。
  4. 结尾钩子：沈砚捡起一枚掉落的"时序残页"碎片——残页上有一行不属于这个时间线的字：林叙白的笔迹。
- **本章核心爽点**：信息差碾压 + 公开式打脸灭口者 + 残本机制首次显威。
- **本章结尾钩子**：已死的恩师，在另一版时间线里给他留了字。

### 第 4 章：编外

- **本章目标**：官方介入，苏鸢登场，主角半入局；引出锚管局设定。
- **出场人物**：沈砚、苏鸢。
- **埋伏笔**：锚定仪监测到桥面时间裂缝；局内鬼暗线（某高层过快介入）。
- **收伏笔**：第 3 章灭口者退走留下的"蚀刻会"之名（被苏鸢确认是真实组织）。
- **剧情流**：
  1. 开篇桥：沈砚持残页回到事务所，被发现出现在锚管局监测名单上。
  2. 冲突发展：苏鸢以"外勤特管员"身份上门评估，判定沈砚为"1–2 级潜流，低危"，编外协查。
  3. 爽点落地：沈砚反过来用残页信息试探苏鸢——"你们局里有人在等这次裂缝。"苏鸢沉默，半信任关系建立。
  4. 结尾钩子：苏鸢离开后接到局里电话："刻钟已经盯上他了。"
- **本章核心爽点**：扮猪吃虎（被低估）+ 与官方半信任的智斗张力。
- **本章结尾钩子**：刻钟出场预告。

### 第 5 章：黑市

- **本章目标**：姜十七登场，提供地下信息源；铺"刻钟是打工人"动机。
- **出场人物**：沈砚、姜十七。
- **埋伏笔**：姜十七与沈砚"互相欠过命"的旧账（卷一中段揭）；黑市能买到"伪异能药剂"。
- **收伏笔**：第 3 章残页上的林叙白笔迹（被姜十七认出是"时序残页"）。
- **剧情流**：
  1. 开篇桥：沈砚需要搞清"时序残页"是什么，官方问不了，只能找黑市。
  2. 冲突发展：姜十七登场，嘴碎滑头，认钱不认人，却报了个异常低的价格——"就当还你那一命。"
  3. 爽点落地：沈砚用残本"读"出姜十七藏在抽屉里的真情报，反向加价压回，建立平等灰色盟友关系。
  4. 结尾钩子：姜十七神色一正："刻钟不是冲你来的，他是冲'稿本'来的。你师父，是上一任执笔人候选。"
- **本章核心爽点**：灰色交易里的信息差反制 + 抛出师父身份长线伏笔。
- **本章结尾钩子**：林叙白是"执笔人候选"——师父之死远不是意外。

### 第 6 章：稿本

- **本章目标**：抛出"现实是被反复改写的稿本"核心长线设定；主角第一次面对"救不回"。
- **出场人物**：沈砚、（残本里的）林叙白残影。
- **埋伏笔**："稿本"概念、执笔人、残本不可逆（救不回死人）。
- **收伏笔**：第 5 章姜十七抛出的"执笔人候选"。
- **剧情流**：
  1. 开篇桥：沈砚回到桥面"时间锚点"，试图用残本读取师父坠桥那晚的真相。
  2. 冲突发展：残本里林叙白留下一段话——"现实是一份稿子，有人一直在改。砚儿，别想救我，救不回来。"
  3. 爽点/情感落地：沈砚本能想回溯救人，发现残本规则锁死——已死之人捞不回，只能读信息。第一次直面"总能重来"的代价。
  4. 结尾钩子：残本末尾，林叙白说："你要小心那个叫执笔人的……他也是读残本的。"
- **本章核心爽点**：设定揭示 + 情感痛点（成长痛起点）。
- **本章结尾钩子**：执笔人也是读残本的——双向时间博弈正式成立。

### 第 7 章：刻钟

- **本章目标**：卷一明线反派刻钟正面登场；三轮智斗第一回合。
- **出场人物**：沈砚、刻钟、苏鸢。
- **埋伏笔**：刻钟"打工人"动机（为晋升资源）；刻钟能感知裂缝定位。
- **收伏笔**：第 4 章苏鸢电话"刻钟盯上他了"。
- **剧情流**：
  1. 开篇桥：锚定仪再次报警，沈砚被苏鸢紧急转移。
  2. 冲突发展：刻钟设局在旧工业区"草稿层裂口"逼沈砚现身，第一回合正面交锋。
  3. 爽点落地：沈砚用"最小改动"——仅回溯一次挪位，反将刻钟预判的站位落空；刻钟察觉"这小子比预估的难缠"。
  4. 结尾钩子：刻钟留下一枚"时序残页"当战书："下次，我不会只等你倒带十秒。"
- **本章核心爽点**：正面智斗不落下风 + 反派被迫正视主角。
- **本章结尾钩子**：升级到分钟级回溯的威胁预告。

### 第 8 章：局内

- **本章目标**：局内鬼暗线推进；沈砚用残本做第一次"时间线版本对比"解谜。
- **出场人物**：沈砚、苏鸢、（暗线）局内鬼。
- **埋伏笔**：局内鬼身份线索（某份报告时间戳不对）。
- **收伏笔**：第 4 章某高层过快介入的伏笔。
- **剧情流**：
  1. 开篇桥：苏鸢递来内部档案，沈砚发现一份事故报告的修改时间戳早于事件发生。
  2. 冲突发展：沈砚用残本调取"上一版时间线"的报告原稿，做版本对比——发现被人提前"写好"了结论。
  3. 爽点落地：信息差碾压式指出"这份报告是穿越时间写的"，逼局内高层露出马脚（未明揪，留线）。
  4. 结尾钩子：被点破的高层当晚联系刻钟："加速解决他。"
- **本章核心爽点**：残本"版本对比"机制首次实战 + 智斗官方内鬼。
- **本章结尾钩子**：局内鬼与刻钟合流，危机升级。

### 第 9 章：草稿层

- **本章目标**：旧工业区"草稿层裂口"副本；沈砚与姜十七深入，铺世界观秘境。
- **出场人物**：沈砚、姜十七。
- **埋伏笔**：草稿层裂口是蚀刻会与黑市交汇点；裂口处时间不稳，回溯代价翻倍。
- **收伏笔**：第 7 章刻钟留下的战书残页（指向裂口）。
- **剧情流**：
  1. 开篇桥：为查刻钟背后线索，沈砚与姜十七下到旧工业区地下裂口。
  2. 冲突发展：裂口处时间紊乱，残本读取出现"叠影"，沈砚第一次见到"多条时间线同时显影"。
  3. 爽点落地：沈砚靠"最小改动"在叠影中分辨出真实信息，拿到刻钟下一次行动的坐标。
  4. 结尾钩子：裂口深处，墙上有一行旧字——"执笔人曾在此定稿。"——长线伏笔再埋。
- **本章核心爽点**：秘境探索 + 残本机制在极端环境下的智斗运用。
- **本章结尾钩子**：执笔人曾亲临此处——其存在感逼近。

### 第 10 章：定调

- **本章目标**：第 10 章中爽点——沈砚用残本公开式破解一起"不可能犯罪"，正式在锚管局立威、确立协查地位。
- **出场人物**：沈砚、苏鸢、锚管局同事若干。
- **埋伏笔**：沈砚被局里部分人盯上（既怕又想用）；中段"不可能犯罪"案的死者与蚀刻会有关。
- **收伏笔**：第 8 章时间线版本对比手法的公开化运用。
- **剧情流**：
  1. 开篇桥：锚管局遇一起"密室级不可能犯罪"求助，苏鸢力荐沈砚。
  2. 冲突发展：常规异能侦查陷入死胡同，沈砚被质疑"一个潜流能干什么"。
  3. 爽点落地：沈砚用残本版本对比，当众还原"凶手用了上一版时间线的手法"——公开式打脸质疑者，确立协查地位。
  4. 结尾钩子：案件死者口袋里有一张照片——背景里模糊站着的人，正是刻钟。
- **本章核心爽点**：扮猪吃虎 + 公开式打脸 + 协查地位确立。
- **本章结尾钩子**：刻钟与案死者关联——三轮智斗终局在即。

## 第三级·补：卷一七弧结构总览（第 1–100 章）
