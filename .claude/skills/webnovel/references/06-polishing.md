# Polishing & Optimization Module

## Execution Flow

1. Load the creative memory ledger; verify core settings and character profiles
2. Confirm the polishing request (pacing adjustment / expansion / condensing / error correction / style unification)
3. Polish according to the corresponding rules; do not alter core plot or settings
4. Validate against the polishing standards
5. Output the polished content + revision notes
6. If settings are adjusted, sync to the creative memory ledger

## 1. Pacing Optimization Rules

1. Delete all redundant description that does not advance the plot, develop characters, or deliver satisfaction beats
2. Break large blocks of exposition into fragments and distribute across multiple chapters
3. Move conflict forward — the opening 100 characters must advance the plot
4. Extend the tension build-up before a satisfaction beat to strengthen the emotional payoff
5. Preserve all foreshadowing-related content; never arbitrarily delete foreshadowing details

## 2. Expansion Rules (Without Diluting Quality)

1. Add detail description: action details, environmental details, character micro-expressions
2. Add bystander reactions: reinforce satisfaction beats through observer perspectives
3. Supplement internal monologue: protagonist's psychological shifts and decision logic
4. Add dialogue interactions: convey information and character relationships through conversation

- Note: Expansion must not be filler — all added content must serve plot or emotion. Do not introduce new settings.

## 3. Condensing Rules

1. Delete redundant adjectives and modifiers
2. Merge paragraphs with repetitive expression
3. Skip unimportant transitional scenes — summarize with a single line
4. Preserve core conflicts, satisfaction beats, hooks, and foreshadowing details; condense everything else

## 4. Error Correction & Unification Rules

1. Fix setting contradictions and name/place errors — use the creative memory ledger as the authoritative source
2. Unify person, perspective, and narrative style
3. Fix typos, grammar errors, and awkward sentences
4. Adjust paragraph length to match reading standards
5. Fix OOC content to align with character profiles

## Polish Validation Checklist

- [ ] Core plot and character settings are unchanged; consistent with creative memory
- [ ] Pacing is smoother; satisfaction beats are more prominent
- [ ] Foreshadowing details are fully preserved; nothing lost
- [ ] No new redundant or useless content added
- [ ] No typos or setting contradictions
