# Topic Planning Module

## Execution Flow

1. Confirm genre track and core target audience
2. Generate a core premise using the "Golden Premise Formula"
3. Match core satisfaction beats and emotional value
4. Output a complete premise card
5. Validate against the premise checklist
6. Sync updates to the creative memory ledger's core settings

## The Golden Premise Formula

Good premise = Hot genre track + Core hook + Differentiating twist + Clear satisfaction driver

## Core Satisfaction Beat Library (Match by Genre)

- **Progression Fantasy (升级流 / shēngjí liú)**: Power growth, social mobility, proving doubters wrong
- **System Genre (系统流 / xìtǒng liú)**: Cheat-ability empowerment, quest rewards, accelerated growth
- **Underdog / Comeback (赘婿 / 逆袭 / zhuìxù / nìxí)**: Rising from the bottom, identity reveal, collective shock
- **Kingdom-Building (种田经营 / zhòngtián jīngyíng)**: Building from nothing, wealth accumulation, empire expansion
- **Mystery / Thriller (悬疑解密 / xuányí jiěmì)**: Layer-by-layer twists, truth reveal, intellectual dominance
- **Romance (情感向 / qínggǎn xiàng)**: Mutual pursuit, second-chance reunion, protective bonds
